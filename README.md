# Multi-Level Company Data Scraper

## 📌 Overview
This project is an advanced Python-based web scraping tool designed to extract structured data from complex, multi-level websites. It navigates through hierarchical pages (regions → companies → details) and handles dynamic content such as AJAX loading and pagination.

---

## 🛠️ Technologies Used
- Python
- Selenium
- BeautifulSoup
- CSV / JSON

---

## ⚡ Features
- Multi-level navigation scraping (Region → Company → Details)
- Handles dynamic content (AJAX, infinite scroll)
- Automated browser interaction using Selenium
- Data parsing using BeautifulSoup
- Export data into structured formats (CSV & JSON)
- Scalable and reusable scraping logic
